# py-study

Small Python practice repository used between larger projects to keep problem-solving skills sharp.

## Components

### engineering_math
Utility tasks and scripts focused on simplifying test/problem solving from the book *Engineering Mathematics* by K. A. Stroud and Dexter J. Booth.

- Current focus: polynomial equation solving helpers (iterative integer search and symbolic solving with `sympy`).
- Main file: `engineering_math/polynomials.py`.

### hackerrank_com
Collection of short solutions for HackerRank exercises, including both core Python and NumPy challenges.

- Organized by topic folders such as `challenges`, `closures-and-decorators`, and `numpy-challenge`.
- Most files are standalone scripts that read input and print challenge output.

### pythonprinciples_com
Set of compact exercise solutions from PythonPrinciples, mostly function-oriented beginner/intermediate tasks.

- Examples include string/list utilities and small logic problems (`anagrams`, `palindrome`, `flatten_list`, etc.).
- Several files show multiple solution variants for the same prompt.
